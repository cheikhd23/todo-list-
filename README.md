                                                            📝 To-Do List App

📌 Description

A simple, clean, and interactive To-Do List web application built using HTML, CSS, and JavaScript.
This project allows users to manage daily tasks efficiently with a modern user interface and persistent storage.



🚀 Features
• Add new tasks
• Mark tasks as completed
• Delete tasks
• Save tasks using localStorage (data persists after refresh)
• Responsive and modern design
• Fast and lightweight (no frameworks)

⸻

🛠️ Technologies Used
	•	HTML5 : Structure of the application
	•	CSS3 : Styling and layout
	•	JavaScript (Vanilla JS) : Functionality and logic
	•	Font Awesome :  Icons

🧠 How It Works
	•	The user enters a task in the input field
	•	Clicking the ➕ button adds the task to the list
	•	Clicking a task marks it as completed (toggle)
	•	Clicking ❌ deletes the task
	•	All tasks are stored in the browser using localStorage
